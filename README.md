** 📊 HR Attrition Insight & Risk Identification**

🔍 Project Overview :
         This Power BI dashboard analyzes employee attrition patterns and identifies high-risk employees using HR data. The goal is to help HR teams reduce employee turnover through data-driven decisions.

🎯 Objectives :
         1.Analyze overall attrition rate
         2.Identify high-risk departments
         3.Study impact of salary, job role, experience, and satisfaction
         4.Predict potential employee exit patterns

📂 Dataset Features :
         Employee ID
         Age
         Department
         Job Role
         Salary
         Years at Company
         Job Satisfaction
         Work-Life Balance
         Attrition (Yes/No)

📊 Dashboard Insights :
        1.📈 Attrition Rate by Department
        2.💰 Salary vs Attrition Analysis
        3.📅 Years at Company vs Attrition
        4.😊 Job Satisfaction Impact
        5.🔥 High-Risk Employee Identification

🛠 Tools Used :
        1.Power BI
        2.DAX
        3.Data Cleaning in Power Query
        4.Excel (if used)

📌 Key Findings :
        1.Employees with low job satisfaction have higher attrition
        2.Salary below average increases exit risk
        3.Employees in sales department show higher turnover

🚀 How to Use :
        1.Download the .pbix file
        2.Open using Power BI Desktop
        3.Refresh data if required
